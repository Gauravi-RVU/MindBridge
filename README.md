# MindBridge

## Digital Mental Health Support Platform for Students

MindBridge is a full-stack web application designed to provide accessible, confidential, and scalable mental health support for students. The platform offers self-screening tools, counsellor appointment booking, peer support forums, mental health resources, and institutional analytics.

## Features

- Secure JWT-based Authentication
- Role-Based Access Control (Student, Counsellor, Admin)
- PHQ-9 & GAD-7 Mental Health Screening
- Confidential Counsellor Appointment Booking
- Anonymous Peer Support Forum
- Multi-language Mental Health Resources
- Admin Analytics Dashboard
- Responsive User Interface

## Tech Stack

### Frontend
- React.js

- Context API
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

- JWT Authentication

- Axios

## System Architecture

Frontend (React.js)
→ REST APIs
→ Express.js Backend
→ MongoDB Database

## Modules

### Student
- Register/Login
- Mental Health Self-Screening
- Book Counselling Appointments
- Access Wellness Resources
- Participate in Anonymous Forums

### Counsellor
- Manage Appointments
- Review Screening Results
- Provide Student Support

### Admin
- Monitor Platform Usage
- View Anonymous Analytics
- Manage Resources and Users

## Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/MindBridge.git
cd MindBridge
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd server
npm install
```

### Run Application

Backend:

```bash
npm start
```

Frontend:

```bash
npm start
```
